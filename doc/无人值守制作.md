# 无人值守制作与制造清单联动

微风采用轻量级的单节点机制。一张普通配方最多设置一个无人值守节点，制作到指定进度后会自动停下，半成品按照游戏内绝对时间等待。等待结束后重新激活半成品即可继续制作，超过最迟处理时间则会损坏。

玩家进入无人值守工序时可以原地等待、直接离开、完成时响铃，或提前五分钟响铃。精确的完成与截止时刻需要手表，设置响铃需要带有 `ALARMCLOCK` 功能的手表、手机、闹钟、载具部件或人物能力。设置新的工序闹钟会替换人物当前已有的闹钟。

再次激活等待中的半成品，可以查看剩余时间并重新选择等待或设置闹钟。物品说明会显示等待中、可以继续或已经损坏的状态。

NPC 制造清单会保留进入无人值守阶段的半成品及其原有顺序，但自动跳过尚未到时的项目，继续处理清单中的后续项目。前面的工序到时后，只要制造清单仍在运行，NPC 会按原顺序回来继续。如果清单里的剩余项目全部处于等待状态，当前清单会结束，需要在工序到时后重新开始一次制造清单。

## JSON 格式

```json
"unattended": {
  "start_at": 50,
  "time": "30 m",
  "max_time": "90 m",
  "start_message": "锅里的食物已经开始慢炖，你可以暂时离开。",
  "ready_message": "慢炖已经完成，可以回来继续处理。",
  "failure_message": "食物放得太久，已经焦糊损坏。"
}
```

`start_at` 是进入无人值守工序时的制作百分比，只能填写 1 至 99。为配合工具消耗检查，推荐使用 5 的倍数。

`time` 是必须等待的游戏时间。

`max_time` 是从进入无人值守工序开始计算的最迟处理时间，必须大于 `time`。省略后半成品不会因等待过久而损坏。

三条消息均可省略，省略后使用通用提示。

## 当前示范菜谱

净水：操作至 50%，等待 5 分钟，30 分钟后烧干。

蔬菜高汤：操作至 50%，慢炖 30 分钟，90 分钟后烧坏。

肉汤：操作至 50%，慢炖 30 分钟，90 分钟后烧坏。

面包：操作至 50%，发酵 10 分钟，2 小时后发酵过度。

脱水肉：操作至 50%，脱水 15 分钟，2 小时后烤焦。

## 当前限制

每张配方只支持一个节点，不会持续检查热源、电力与环境条件，无人值守时间不随批量大小变化。闹钟沿用人物现有的单闹钟机制，暂不支持同时为多份半成品设置多个独立提醒。NPC 清单暂不具备远距离自动唤醒，全部项目都在等待时不会原地空转。
